# Task2: Support Ticket Classification & Prioritization System

An end-to-end NLP and machine learning project that classifies support tickets into business categories and predicts ticket priority for triage automation.

## Files

- `task2.ipynb`: main notebook with data generation, preprocessing, training, evaluation, and UI
- `eda_overview.png`: EDA dashboard generated by the notebook
- `confusion_matrices.png`: confusion matrices for category and priority models
- `model_comparison.png`: performance comparison chart

## Project Scope

- Category classification across Billing Issue, Technical Issue, Account Problem, and General Query
- Priority prediction across High, Medium, and Low
- TF-IDF vectorization with NLTK preprocessing
- Logistic Regression, Multinomial Naive Bayes, and Random Forest baselines
- Interactive notebook UI for live ticket triage

## Run

1. Install dependencies used in the notebook: `pandas`, `numpy`, `matplotlib`, `seaborn`, `scikit-learn`, `nltk`, `jinja2`, `ipywidgets`.
2. Open `task2.ipynb` in Jupyter or VS Code.
3. Run the notebook from top to bottom.
4. Use the interactive UI section after the training and prediction cells have executed.

## Notes

The dataset is synthetic and built inside the notebook to mimic realistic support-ticket language while avoiding sensitive customer data.
